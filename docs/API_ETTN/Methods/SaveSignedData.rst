**JSON-параметри в тілі HTTP запиту/відповіді**
*******************************************************************

``REQUEST``

В цьому методі в **запиті** передається підпис (в бінарному вигляді).

.. attention:: Підпис НЕ передається в BASE64 форматі! 

``RESPONSE``

У **відповідь** передається код сервера 200 (ok).
